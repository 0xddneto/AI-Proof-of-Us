# AI Proof of Us

AI Proof of Us, or AIPOU, is an open experiment for **portable receipts for AI-assisted work**.

Instead of asking every model, app, or marketplace to invent its own proof layer, AIPOU gives agents a shared flow:

```text
wallet authorization -> task nonce -> AI work -> signed receipt -> validator checks -> Merkle claim -> AIPOU on Base
```

The goal is simple: if useful AI work happens across Codex, Claude, Cursor, OpenClaw, local models, or future MCP clients, the user should be able to keep a privacy-preserving receipt and later claim approved rewards.
